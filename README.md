# Remote Worker Geo-Arbitrage Cost-of-Living Modeler

A lightweight, standalone web application that allows remote professionals to calculate the financial impact of geographical arbitrage.

## 🚀 Architectural & Core Features
- **Hotspot Dataset Indexing**: Features multi-region expense baselines across highly searched nomad hubs (Southeast Asia, Europe, Latin America).
- **Compounding Surplus Projections**: Automatically evaluates raw monthly differences and extrapolates a 12-month compounding savings timeline.
- **Zero-Dependency Core**: Completely self-contained script architecture optimizing browser rendering efficiency for maximum responsive speeds.
